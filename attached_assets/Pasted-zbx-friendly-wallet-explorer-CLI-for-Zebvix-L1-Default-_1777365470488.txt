zbx — friendly wallet/explorer CLI for Zebvix L1.

Default RPC: http://127.0.0.1:8545 (override with --rpc or env ZBX_RPC)

Usage: zbx [OPTIONS] <COMMAND>

Commands:
  new      Create a brand new wallet (keypair) and save to file
  show     Show address & pubkey of an existing keyfile
  address  Print just the address of a keyfile (alias of `show`, address only)
  import   Import a wallet from a raw 64-char hex secret key
  balance  Get ZBX balance of an address
  nonce    Get current pending nonce of an address
  send     Send ZBX from a wallet to an address
  tip      Show chain tip (latest block height & hash)
  info     Show node info: chain id, block time, supply
  block    Get a block by height
  supply   Show ZBX supply summary (total / circulating / pre-mine)
  status   Show node sync/peer status
  pool     Show on-chain AMM pool state (ZBX/zUSD reserves, price, fee, loan, fees)
  swap     Swap ZBX → zUSD by sending to the pool address (one-shot helper). Equivalent to `zbx send --to <POOL_ADDRESS> --amount <N>`
  price    Show current ZBX/USD price (from on-chain pool oracle)
  gas      Show network's current dynamic gas price + fee per standard transfer
  zusd     Show zUSD balance of an address
  lp       Show LP (liquidity-provider) token balance of an address
  admin    Show admin/founder address info (current address, rotations used, max)
  help     Faucet-style helper: print common testnet info
  help     Print this message or the help of the given subcommand(s)

Options:
      --rpc <RPC>
          JSON-RPC endpoint of a running Zebvix node
          
          [env: ZBX_RPC=]
          [default: http://127.0.0.1:8545]

  -q, --quiet
          Quiet output — JSON only, no colors or banners

  -h, --help
